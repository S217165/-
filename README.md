
本程序使用scrapy架构，共采集6页李开复博客目录与252篇博文信息，实现了数据清洗处理，在控制台输出问题数据，并将修改后的数据分别存入MongoDB数据库和csv文件，其间有进度条显示程序运行进度。

使用方法：运行main.py
